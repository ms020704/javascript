To-Do 리스트 앱

설명: 할 일을 추가, 완료 체크, 삭제하는 간단한 To-Do List 웹앱입니다.
주요 기능:
  • 할 일 추가/삭제/완료 체크
  • 로컬스토리지로 데이터 저장
  • 완료시 줄긋기, 삭제 버튼
사용 기술: HTML, CSS, JavaScript(LocalStorage)
느낀 점/문제 해결: 처음엔 할 일을 새로고침하면 사라졌으나, 로컬스토리지를 배우고 데이터가 유지되도록 개선하였습니다.
삭제 버튼 클릭 시 부모 li 이벤트와 충돌하는 문제를 e.stopPropagation()으로 해결함.
